Pobranie i zainstalowanie obrazów dla dockera#Pliki i folderydocker/.env - nazwa kontenera na serwisydocker/docker-compose.yml - pobranie i instalacja obrazówdocker.sh - skrypt pobieraj¹cy, instaluj¹cy i uruchamiajacy wszystkie serwisy w dockerdocker-stop.sh - zatrzymuje wszystkie serwisyInstrukcja [ubuntu]1. Skopiuj folder "Docker" i pliki *.sh2. sudo chmod 777 docker.sh3. sudo chmod 777 docker-stop.sh4. sudo ./docker.sh
